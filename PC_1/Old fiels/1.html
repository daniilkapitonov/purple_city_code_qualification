Для начала давайте разберемся в функционале Atom (при работе с Notepad++ никаких проблем не будет, можно сразу начать программировать)

Для начала работы в Atom:

  1. Cоздать новый файл
  2. Справа снизу найти надпись Plain Text, нажать
  3. В строке поиска ввести html и выбрать первый результат

    4. Что такое HTML?

    HTML(HYPER TEXT MARKUP LANGUAGE)-стандартизированный язык разметки документов во Всемирной паутине. То есть это способ отобразить основные части документа на web-страницах. Этот основной язык программирования-скелет страницы.В таком случае Java Script- мышцы (он создает кнопочки и тд), C++ - лицо,кожа.

    Структура HTML-документа

    Какие теги бывают?

    Теги бывают открывающими-<...>, закрывающими- . Пример: открывающий тег должен быть перед текстом, который мы хотим изменить , а закрывающий после. Если я в уроке пишу, допустим, тег B, то это значит, что его надо писать вместо троиточия.

    Атрибуты

    Атрибуты-это параметры простым языком. Мы пишем их "после троеточия" и ставим после них = и "", в "" мы пишем значение атрибута.

    PRE

    Изучим основной тег языка-PRE. Он используется для записи текста. Он сохраняет текст точно также, как мы его написали. Посмотрите на картинку снизу. Это код части этой страницы. Напишите его и убедитесь.

    Запустите(сохраните файл с расширением html и откройте этот файл в браузере). Если это выглядит не так, как на картинке, перепроверьте код
